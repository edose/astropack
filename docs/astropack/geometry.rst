###############################################################
Point and Vectors classes for Photometry (`astropack.geometry`)
###############################################################

Classes |XY| (point) and |DXY| (vector) for managing locations in a 2-dimensional plane
as for photometric apertures in an image.

Classes |Rectangle_in_2D| and |Circle_in_2D| for defining regions in a 2-dimensional plane
as for constructing aperture masks for background-corrected aperture photometry.
